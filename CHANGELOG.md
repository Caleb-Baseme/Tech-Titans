## [0.2.0] - 2026-07-15

### Added
- User Profile screen.
- Navigation icons.
- Forgot passwprd option.
- Walking distance indicator.
- Parking History preview.

### Changed
- Redesigned Home Dashboard using a medium-fidely layout.
- Enlarged the 'Save My Spot' button.
- Improved spacing and typography.
- Replaced placeholders with realistic interface elements.
- Updated 'Nearby Parking' with card-style results.
- Improved 'Parking Timer' with a larger countdown timer.

### Fixed
- Corrected inconsistent navigation.
- Standardized button styles.
- Improved readibility.
- Fixed layout alignment issues.

## [0.1.0] - 2026-07-08

### Added
- Created the first low-fidelity wireframe for the ParkPal mobile application.
- Added Login, Home Dashboard, Save My Spot, Parking Timer, Nearby Parking Search, Saved Parking Location, and navigation.

### Changed
- Reorganized the Home Dashboard to make 'Save My Spot' the primary action.
- Simplified navigation and standardized layouts.

### Fixed
- Corrected navigation links.
- Fixed inconsistent labels and alignment.
- Ensured all screens follow the agreed black-and-white low-fidelity standard.
